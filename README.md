## 🌍 The World of Aeloria

**Aeloria** (艾洛瑞亚大陆) is an original fantasy continent home to five regions and five races, where ancient nature magic clashes with industrial technology.

### Main Character — Toron, the Silver Armor Totoro
A legendary forest guardian beast who slept for a thousand years, sealed inside ancient roots, now awakened. His silver armor — crafted by the Sylari healer Silvi — channels the life force of the continent itself.

### The Five Regions

| Region | Theme | Key Feature |
|--------|-------|-------------|
| 🌿 Lingsen Domain (West) | Ancient forest | Home of Toron, glowing sacred trees |
| ⚒️ Iron Plateau (North) | Industrial empire | Steam-powered cities, iron mines |
| 🌊 Blue Tide Isles (East) | Ocean archipelago | Seafaring traders, coral ruins |
| ☀️ Red Sand Empire (South) | Desert civilization | Fossil magic, ancient stone cities |
| ❄️ Frost Wastes (Arctic) | Frozen wilderness | Dormant monsters, lost ruins |

## 🎯 How to Use

**All files are self-contained HTML — no install needed.**

1. Clone or download this repository
2. Open `index.html` in any browser
3. Click any project card to launch it

```bash
git clone https://github.com/longmaolab/aeloria-ai-project.git
cd aeloria-ai-project
open index.html   # macOS
# or double-click index.html in Finder / Explorer
```

To publish via GitHub Pages:
1. Go to **Settings → Pages**
2. Set source to `main` branch, root folder
3. Your site will be live at `https://longmaolab.github.io/aeloria-ai-project`
